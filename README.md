# plot-extraction
Extract plot data from papers. 

This tool is inspired by Plot2Spectra, but modernized with the ability to automatically extract figures from papers (via Claude Code backend). 

Built with the intention to help automate efforts in paper replication (specifically for papers describing integrated silicon photonic devices).


For fun and out of the author's curiosity, Plot2Spectra was reimplemented from scratch (I may have messed up the algorithm, that remains to be seen).
